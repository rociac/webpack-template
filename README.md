## Installation

1. Clone the repository in your local folder.
```
git clone https://github.com/rociac/webpack-template.git
```
2. Run
```
npm install
```
3. To run the development server
```
npm start
```
4. To make a production build (creates dist folder)
```
npm run build
```
5. To run tests
```
npm test
```
